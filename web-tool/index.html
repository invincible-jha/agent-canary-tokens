How It Works

  1. Click "Generate Token" to create a unique canary string.
  2. Copy the string and embed it in your documents or content.
  3. If you later encounter the exact string in AI output, your content was ingested.
  4. Use the "Check" feature to see if others have reported sightings of your token.

No personal data is stored. Only random token identifiers are kept on the server. See the project documentation for full privacy details.
